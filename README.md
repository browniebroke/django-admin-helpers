A collections of helpers for the Django admin.

## Installation

Install this via pip (or your favourite package manager):

`pip install django-admin-helpers`

Add the app to your `INSTALLED_APPS`:

```python
INSTALLED_APPS = [
    # ...
    "django_admin_helpers",
]
```
